Introduction

This task is designed to resemble our frontend development environment and get you working on some realistic dummy data.

We are interested in the price history of company bonds. There is some raw data available from two API endpoints, which for this task are provided as files holding JSON data. A brief description of these files is provided after the requirements are defined. Please feel free to contact us if any aspects of the requirements are unclear.

Requirements

The system...
  • MUST display a list of the companies in the companies.json dataset.
  • MUST provide a view for each company, displaying their basic info and details on their relevant bonds.
  • SHOULD display the history of the price of each bond.
  • COULD show a list of the highest five moving bond prices. This means the difference between the opening and closing bond price for a particular time period.

JSON file description

Two files are provided representing the output from two API services:
  • app/json/companies.json
  • app/json/bonds.json
